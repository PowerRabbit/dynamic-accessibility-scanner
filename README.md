## Getting Started

### Install the dependencies
```bash
npm i
```

### Build & Run

```bash
npm run build
```
```bash
npm start
```

### OR run in development mode

```bash
npm run dev
```

### Open

Open [http://localhost:3000](http://localhost:3000) with your browser.
